# 🎬 EverythingMovies: AI-Powered Entertainment Analytics

EverythingMovies is a full-stack web application designed for seamless movie discovery. It combines traditional streaming platform aggregation with modern AI recommendation engines.

🚀 Key Features
* **AI Recommendation Engine**: Find similar movies based on content analysis.
* **Rating Predictor**: Predict user scores using ML logic.
* **Google OAuth 2.0**: Secure user authentication.
* **Platform Insights**: Real-time pros/cons for streaming sites.

🛠️ Tech Stack
* **Frontend**: Streamlit
* **Backend**: FastAPI / Python
* **APIs**: TMDB API, Clearbit Logo API
* **Auth**: Google Cloud Console (OAuth 2.0)

📥 Installation
1. Clone the repo: `git clone https://github.com/your-username/repo-name.git`
2. Install dependencies: `pip install -r requirements.txt`
3. cd EverythingMovies-AI-Analytics/entertainment-analytics
4. Run the app: `streamlit run app.py`


📋 Prerequisites
- Python 3.9 or higher
- A Google Cloud Project with OAuth 2.0 enabled
